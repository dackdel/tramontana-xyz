what is tramontana?

tramontana leverages the capabilities of the object that we have all come to carry with us anywhere, all the time, our smartphones. With libraries for Processing, Javascript and OpenFrameworks you can access the inputs and outputs of one or more smartphones to easily and quickly prototype interactive spaces, connected products or just something youâ€™d like to be wireless. What used to involve complex tasks like networking, native app development, etc. can now be created with a single sketch on your computer.

adding the library:
1. After you have downloaded & installed processing, open it
2. In the menu bar click on Sketch > Import Library > Add library
3. Search for 'Tramontana' and click Install
4. Install 'Websockets' in the same way
